## Key Takeaways

✅ **React Native components work in Fumadocs** through React Native Web
✅ **Adaptive components** like Modal work on both web and native
✅ **Platform detection** via `Platform.OS` enables conditional rendering
✅ **Portal** renders outside normal DOM tree (modals, tooltips, etc.)
✅ **Flexbox** layout system is fully compatible
✅ **Interactive state** works with React hooks

## Code Example: Using Adaptive Modal

```tsx
import { AdaptiveModal, Portal } from '@lunar-primitive/adaptive-modal';
import { Pressable, Text, View } from 'react-native';

export function MyComponent() {
  const [visible, setVisible] = useState(false);

  return (
    <View>
      <Pressable onPress={() => setVisible(true)}>
        <Text>Open Modal</Text>
      </Pressable>

      <AdaptiveModal
        visible={visible}
        onDismiss={() => setVisible(false)}
      >
        <View>
          <Text>This works on iOS, Android, AND Web!</Text>
        </View>
      </AdaptiveModal>
    </View>
  );
}
```
