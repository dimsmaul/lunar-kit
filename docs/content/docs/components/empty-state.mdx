A component for displaying an empty state with an optional illustration, title, description, and action button.

<EmptyStateDemo />

## Installation

<DynamicCodeBlock lang="bash" code={`lunar add empty-state`} />

## Usage

<DynamicCodeBlock lang="tsx" code={`import { EmptyState } from '@/lunar-kit/components/empty-state';`} />

### Basic Usage

<DynamicCodeBlock lang="tsx" code={`<EmptyState
  title="No results found"
  description="Try adjusting your search or filter to find what you're looking for."
/>`} />

### With Action Button

<DynamicCodeBlock lang="tsx" code={`<EmptyState
  title="No items yet"
  description="Get started by adding your first item."
  actionLabel="Add Item"
  onActionPress={() => handleAdd()}
/>`} />

### With Custom Illustration

<DynamicCodeBlock lang="tsx" code={`import { PackageOpen } from 'lucide-react-native';

<EmptyState
  title="Your cart is empty"
  description="Browse our products and add items to your cart."
  illustration={<PackageOpen size={48} color="#94a3b8" />}
  actionLabel="Browse Products"
  onActionPress={() => navigateToProducts()}
/>`} />

## API Reference

### EmptyState

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | Required | Title text |
| `description` | `string` | — | Description text |
| `illustration` | `ReactNode` | — | Custom illustration/icon |
| `actionLabel` | `string` | — | Button label (shows button when provided) |
| `onActionPress` | `() => void` | — | Button press handler |
| `className` | `string` | — | Additional classes |
