A skeleton loading placeholder component with pulsing animation and multiple shape variants.

<SkeletonDemo />

## Installation

<DynamicCodeBlock lang="bash" code={`lunar add skeleton`} />

## Usage

<DynamicCodeBlock lang="tsx" code={`import { Skeleton } from '@/lunar-kit/components/skeleton';`} />

### Rect (Default)

<DynamicCodeBlock lang="tsx" code={`<Skeleton className="h-24 w-full" />`} />

### Circle

<DynamicCodeBlock lang="tsx" code={`<Skeleton variant="circle" className="w-12 h-12" />`} />

### Text

<DynamicCodeBlock lang="tsx" code={`<Skeleton variant="text" />
<Skeleton variant="text" className="w-3/4" />`} />

### Loading Card

<DynamicCodeBlock lang="tsx" code={`<View className="gap-4 p-4">
  <View className="flex-row gap-4 items-center">
    <Skeleton variant="circle" className="w-12 h-12" />
    <View className="flex-1 gap-2">
      <Skeleton variant="text" className="w-1/2" />
      <Skeleton variant="text" className="w-3/4" />
    </View>
  </View>
  <Skeleton className="h-40 w-full" />
  <View className="flex-row gap-2">
    <Skeleton className="h-10 w-20 rounded-lg" />
    <Skeleton className="h-10 w-20 rounded-lg" />
  </View>
</View>`} />

## API Reference

### Skeleton

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'rect' \| 'circle' \| 'text'` | `'rect'` | Shape variant |
| `className` | `string` | — | Additional classes |
