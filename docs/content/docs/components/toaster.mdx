Toaster is a container component that renders all active toast notifications. Place it in your root layout to display toasts anywhere in your app.

<ToasterDemo />

## Installation

<DynamicCodeBlock lang="bash" code={`lunar add toaster`} />

## Setup

Add `<Toaster />` to your root layout or main app component:

<DynamicCodeBlock lang="tsx" code={`import { Toaster } from '@lunar-kit/core';

export default function RootLayout({ children }) {
  return (
    <>
      {children}
      <Toaster />
    </>
  );
}`} />

## Usage

Once Toaster is placed in your root layout, use `toast()` function anywhere in your app:

<DynamicCodeBlock lang="tsx" code={`import { Button, toast } from '@lunar-kit/core';

export function MyComponent() {
  return (
    <Button
      onPress={() => toast.success('Success!', 'Done')}
    >
      Show Toast
    </Button>
  );
}`} />

## How It Works

1. `<Toaster />` listens to the toast store
2. Call `toast()` from any component
3. Toaster automatically renders and removes toasts
4. Toasts display at top or bottom with animations
