## API Reference

### Separator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Orientation of the separator |
| `decorative` | `boolean` | `true` | If false, adds `aria-role="separator"` |
| `className` | `string` | — | Additional classes |
