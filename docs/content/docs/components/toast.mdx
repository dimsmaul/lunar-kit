A toast notification system with swipe-to-dismiss, multiple variants, positioning, and an imperative API.

## Installation

<DynamicCodeBlock lang="bash" code={`lunar add toast`} />

## Setup

Add the `Toaster` component to your root layout:

<DynamicCodeBlock lang="tsx" code={`import { Toaster } from '@lunar-kit/core';

export default function Layout() {
  return (
    <>
      {/* Your app content */}
      <Toaster />
    </>
  );
}`} />

## Usage

The toast system uses an imperative API via the `toast` function:

<DynamicCodeBlock lang="tsx" code={`import { toast } from '@lunar-kit/core';`} />

### Default

<DynamicCodeBlock lang="tsx" code={`toast('This is a default message')`} />

### Variants

<DynamicCodeBlock lang="tsx" code={`toast.success('Success!', 'Your changes have been saved.')
toast.error('Error', 'Something went wrong.')
toast.warning('Warning', 'Please review your information.')
toast.info('Info', 'Here is some useful information.')`} />

### Custom Position

<DynamicCodeBlock lang="tsx" code={`toast('Bottom toast', { position: 'bottom' })
toast('Top toast', { position: 'top' })`} />

### Custom Duration

<DynamicCodeBlock lang="tsx" code={`toast('Short toast', { duration: 2000 })
toast('Long toast', { duration: 5000 })`} />

### Direction

Control the entry animation direction:

<DynamicCodeBlock lang="tsx" code={`toast('Slide from left', { direction: 'left' })
toast('Slide from bottom', { direction: 'bottom' })`} />

## API Reference

### toast()

The imperative toast function:

<DynamicCodeBlock lang="tsx" code={`toast(props: ToastProps)`} />

Shorthand methods:

<DynamicCodeBlock lang="tsx" code={`toast.success(title, description?, options?)
toast.error(title, description?, options?)
toast.warning(title, description?, options?)
toast.info(title, description?, options?)`} />

### Toast Options

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | — | Toast title |
| `description` | `string` | — | Toast description |
| `variant` | `'default' \| 'success' \| 'error' \| 'warning' \| 'info'` | `'default'` | Visual variant |
| `duration` | `number` | `3000` | Auto-dismiss duration in ms |
| `position` | `'top' \| 'bottom'` | `'top'` | Toast position |
| `direction` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'top'` | Entry animation direction |

### Toaster

The container component that renders all active toasts. Must be added to your root layout.

<DynamicCodeBlock lang="tsx" code={`<Toaster />`} />

### useToastStore

Zustand store hook for advanced usage:

const { toasts, addToast, removeToast, dismissToast } = useToastStore();`} />
