Toast is a function-based notification system. Call `toast()` with a message to show notifications.

<ToastDemo />

## Installation

<DynamicCodeBlock lang="bash" code={`lunar add toast`} />

## Usage

Toast is called as a function, typically in button `onPress` handlers or other interactions:

<DynamicCodeBlock lang="tsx" code={`import { Button, toast } from '@lunar-kit/core';

export default function Example() {
  return (
    <Button
      onPress={() => toast({
        title: 'Success',
        description: 'Operation completed'
      })}
    >
      Show Toast
    </Button>
  );
}`} />

## Examples

### Basic Toast

<DynamicCodeBlock lang="tsx" code={`toast({
  title: 'Hello',
  description: 'This is a message'
})`} />

### Toast Variants

<DynamicCodeBlock lang="tsx" code={`// Success
toast.success('Success!', 'Operation completed')

// Error
toast.error('Error!', 'Something went wrong')

// Warning
toast.warning('Warning!', 'Please check your input')

// Info
toast.info('Info', 'Here is some information')`} />

### With Position

<DynamicCodeBlock lang="tsx" code={`toast({
  title: 'Message',
  position: 'top' // 'top' | 'bottom'
})

toast({
  title: 'Message',
  position: 'bottom',
  direction: 'left' // 'left' | 'right' | 'top' | 'bottom'
})`} />

## API

**toast(config)** - Show default toast
- `title?: string` - Toast title
- `description?: string` - Toast description
- `position?: 'top' | 'bottom'` - Position on screen
- `direction?: 'left' | 'right' | 'top' | 'bottom'` - Entry direction

**toast.success(title, description, config)** - Success variant

**toast.error(title, description, config)** - Error variant

**toast.warning(title, description, config)** - Warning variant

**toast.info(title, description, config)** - Info variant
