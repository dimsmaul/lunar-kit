## Usage

<DynamicCodeBlock lang="tsx" code={`import { Input } from '@/components/ui/input';`} />

### Basic Usage

<DynamicCodeBlock lang="tsx" code={`const [value, setValue] = useState('');

<Input
  value={value}
  onChangeText={setValue}
  placeholder="Enter your name"
/>`} />

### Variants

<DynamicCodeBlock lang="tsx" code={`// Outline (default)

### Sizes

<DynamicCodeBlock lang="tsx" code={`<Input size="sm" placeholder="Small" />
<Input size="md" placeholder="Medium (default)" />

### With Prefix/Suffix

<DynamicCodeBlock lang="tsx" code={`import { Mail, Eye, Search, DollarSign } from 'lucide-react-native';

// Prefix icon
<Input
  placeholder="Email"
  prefix={<Mail size={18} color="#6b7280" />}
/>

// Suffix icon
<Input
  placeholder="Password"
  secureTextEntry
  suffix={<Eye size={18} color="#6b7280" />}
/>

// Both
<Input
  placeholder="Search..."
  prefix={<Search size={18} color="#6b7280" />}
  suffix={<Text className="text-muted-foreground text-sm">⌘K</Text>}
/>

// With text prefix
<Input
  placeholder="0.00"
  prefix={<DollarSign size={18} color="#6b7280" />}
/>`} />

## API Reference

### Input

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'outline' \| 'underline'` | `'outline'` | Visual variant |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Input size |
| `prefix` | `ReactNode` | — | Left side adornment |
| `suffix` | `ReactNode` | — | Right side adornment |
| `error` | `boolean` | `false` | Show error state |
| `containerClassName` | `string` | — | Classes for container |
| `inputClassName` | `string` | — | Classes for input |
