## Usage

<DynamicCodeBlock lang="tsx" code={`import { Checkbox, CheckboxLabel, CheckboxDescription } from '@/components/ui/checkbox';`} />

### Basic Usage

<DynamicCodeBlock lang="tsx" code={`const [checked, setChecked] = useState(false);

<Checkbox checked={checked} onCheckedChange={setChecked} />`} />

### With Label

<DynamicCodeBlock lang="tsx" code={`<Checkbox checked={checked} onCheckedChange={setChecked}>
  <CheckboxLabel>Accept terms and conditions</CheckboxLabel>
</Checkbox>`} />

### Indeterminate State

For "select all" scenarios:

<DynamicCodeBlock lang="tsx" code={`const [items, setItems] = useState([
  { id: 1, checked: true },
  { id: 2, checked: false },
  { id: 3, checked: true },
]);

const allChecked = items.every(item => item.checked);
const someChecked = items.some(item => item.checked);

<Checkbox
  checked={allChecked}
  indeterminate={someChecked && !allChecked}
  onCheckedChange={(checked) => {
    setItems(items.map(item => ({ ...item, checked })));
  }}
>
  <CheckboxLabel>Select All</CheckboxLabel>
</Checkbox>`} />

### Disabled State

<DynamicCodeBlock lang="tsx" code={`<Checkbox disabled checked={false}>

## API Reference

### Checkbox

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | `false` | Whether checkbox is checked |
| `onCheckedChange` | `(checked: boolean) => void` | — | Callback when state changes |
| `children` | `ReactNode` | — | Label content |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Checkbox size |
| `disabled` | `boolean` | `false` | Disable the checkbox |
| `indeterminate` | `boolean` | `false` | Show indeterminate state |
| `className` | `string` | — | Additional classes |

### CheckboxLabel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | Required | Label text |
| `className` | `string` | — | Additional classes |

### CheckboxDescription

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | Required | Description text |
| `className` | `string` | — | Additional classes |
