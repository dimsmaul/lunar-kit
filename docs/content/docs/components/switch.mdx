A toggle switch component with smooth spring animations, controlled/uncontrolled support, and a disabled state.

<SwitchDemo />

## Installation

<DynamicCodeBlock lang="bash" code={`lunar add switch`} />

## Usage

<DynamicCodeBlock lang="tsx" code={`import { Switch } from '@/lunar-kit/components/switch';`} />

### Controlled

<DynamicCodeBlock lang="tsx" code={`const [checked, setChecked] = React.useState(false);

<Switch
  checked={checked}
  onCheckedChange={setChecked}
/>`} />

### Uncontrolled

<DynamicCodeBlock lang="tsx" code={`<Switch defaultChecked={false} onCheckedChange={(val) => console.log(val)} />`} />

### Disabled

<DynamicCodeBlock lang="tsx" code={`<Switch disabled />`} />

## API Reference

### Switch

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | — | Controlled checked state |
| `defaultChecked` | `boolean` | `false` | Default checked state (uncontrolled) |
| `onCheckedChange` | `(checked: boolean) => void` | — | Callback when checked state changes |
| `disabled` | `boolean` | `false` | Disable the switch |
| `className` | `string` | — | Additional classes for the track |
| `thumbClassName` | `string` | — | Additional classes for the thumb |
