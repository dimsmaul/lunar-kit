A horizontal scrolling carousel component with snap-to-item behavior and pagination dots.

<CarouselDemo />

## Installation

<DynamicCodeBlock lang="bash" code={`lunar add carousel`} />

## Usage

<DynamicCodeBlock lang="tsx" code={`import { Carousel } from '@/lunar-kit/components/carousel';`} />

### Basic Usage

<DynamicCodeBlock lang="tsx" code={`const data = [
  { id: '1', title: 'Item 1' },
  { id: '2', title: 'Item 2' },
  { id: '3', title: 'Item 3' },
];

<Carousel
  data={data}
  renderItem={({ item }) => (
    <View className="bg-primary rounded-xl p-6 items-center justify-center h-40">
      <Text className="text-white font-bold">{item.title}</Text>
    </View>
  )}
/>`} />

### Custom Item Width

<DynamicCodeBlock lang="tsx" code={`<Carousel
  data={data}
  renderItem={({ item }) => <MyCard item={item} />}
  itemWidth={200}
  gap={16}
/>`} />

### Without Dots

<DynamicCodeBlock lang="tsx" code={`<Carousel
  data={data}
  renderItem={({ item }) => <MyCard item={item} />}
  showDots={false}
/>`} />

## API Reference

### Carousel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `T[]` | Required | Array of data items |
| `renderItem` | `({ item, index }) => ReactElement` | Required | Render function for each item |
| `itemWidth` | `number` | `containerWidth - 64` | Width of each item |
| `gap` | `number` | `12` | Gap between items |
| `showDots` | `boolean` | `true` | Show pagination dots |
| `className` | `string` | — | Additional classes for FlatList |
| `containerClassName` | `string` | — | Additional classes for container |
