A one-time password input component with animated cursor, multiple input types, and controlled/uncontrolled support.

<InputOTPDemo />

## Installation

<DynamicCodeBlock lang="bash" code={`lunar add input-otp`} />

## Usage

<DynamicCodeBlock lang="tsx" code={`import { InputOTP } from '@/lunar-kit/components/input-otp';`} />

### Basic Usage

<DynamicCodeBlock lang="tsx" code={`const [value, setValue] = React.useState('');

<InputOTP
  maxLength={6}
  value={value}
  onValueChange={setValue}
/>`} />

### Password Mode

Hide entered characters with bullet points:

<DynamicCodeBlock lang="tsx" code={`<InputOTP
  maxLength={4}
  inputType="password"
  value={value}
  onValueChange={setValue}
/>`} />

### Uncontrolled

<DynamicCodeBlock lang="tsx" code={`<InputOTP
  maxLength={4}
  defaultValue=""
  onValueChange={(val) => console.log(val)}
/>`} />

### Disabled

<DynamicCodeBlock lang="tsx" code={`<InputOTP
  maxLength={6}
  disabled
  value="123456"
/>`} />

## API Reference

### InputOTP

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled value |
| `defaultValue` | `string` | `''` | Default value (uncontrolled) |
| `maxLength` | `number` | `4` | Number of input cells |
| `onValueChange` | `(value: string) => void` | — | Value change callback |
| `disabled` | `boolean` | `false` | Disable the input |
| `className` | `string` | — | Additional classes per cell |
| `containerClassName` | `string` | — | Additional classes for container |
| `inputType` | `'numeric' \| 'default' \| 'password'` | `'numeric'` | Input type behavior |
