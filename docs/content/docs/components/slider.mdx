A slider component with pan and tap gesture support, spring animations, and configurable min/max/step values.

<SliderDemo />

## Installation

<DynamicCodeBlock lang="bash" code={`lunar add slider`} />

## Usage

<DynamicCodeBlock lang="tsx" code={`import { Slider } from '@/lunar-kit/components/slider';`} />

### Controlled

<DynamicCodeBlock lang="tsx" code={`const [value, setValue] = React.useState(50);

<Slider
  value={value}
  onValueChange={setValue}
/>`} />

### Uncontrolled

<DynamicCodeBlock lang="tsx" code={`<Slider defaultValue={25} onValueChange={(v) => console.log(v)} />`} />

### With Min/Max/Step

<DynamicCodeBlock lang="tsx" code={`<Slider
  value={value}
  onValueChange={setValue}
  min={0}
  max={200}
  step={10}
/>`} />

### Disabled

<DynamicCodeBlock lang="tsx" code={`<Slider value={50} disabled />`} />

## API Reference

### Slider

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | — | Controlled value |
| `defaultValue` | `number` | `0` | Default value (uncontrolled) |
| `min` | `number` | `0` | Minimum value |
| `max` | `number` | `100` | Maximum value |
| `step` | `number` | `1` | Step increment |
| `onValueChange` | `(value: number) => void` | — | Value change callback |
| `className` | `string` | — | Additional classes |
| `disabled` | `boolean` | `false` | Disable the slider |
