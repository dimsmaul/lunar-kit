A navigation component that shows the current location within a hierarchy, with clickable links to navigate back.

<BreadcrumbDemo />

## Installation

<DynamicCodeBlock lang="bash" code={`lunar add breadcrumb`} />

## Usage

<DynamicCodeBlock lang="tsx" code={`import { Breadcrumb } from '@/lunar-kit/components/breadcrumb';`} />

### Basic Usage

<DynamicCodeBlock lang="tsx" code={`<Breadcrumb
  items={[
    { label: 'Home', onPress: () => {} },
    { label: 'Products', onPress: () => {} },
    { label: 'Electronics' },
  ]}
/>`} />

### Custom Separator

<DynamicCodeBlock lang="tsx" code={`import { Slash } from 'lucide-react-native';

<Breadcrumb
  items={[
    { label: 'Home', onPress: () => {} },
    { label: 'Settings' },
  ]}
  separator={<Slash size={14} color="#999" />}
/>`} />

### With Active State

The last item is automatically marked as active. You can also manually set an item as active:

<DynamicCodeBlock lang="tsx" code={`<Breadcrumb
  items={[
    { label: 'Home', onPress: () => {} },
    { label: 'Products', onPress: () => {}, active: false },
    { label: 'Electronics', active: true },
  ]}
/>`} />

## API Reference

### Breadcrumb

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `BreadcrumbItem[]` | Required | Array of breadcrumb items |
| `separator` | `ReactNode` | `<ChevronRight>` | Custom separator element |
| `className` | `string` | — | Additional classes |

### BreadcrumbItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | Required | Display text |
| `onPress` | `() => void` | — | Press handler (clickable items) |
| `active` | `boolean` | `false` | Whether item is active (last item auto-active) |
