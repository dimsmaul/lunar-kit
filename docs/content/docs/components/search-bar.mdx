A search input component with a built-in search icon, clear button, and optional debounced value change.

<SearchBarDemo />

## Installation

<DynamicCodeBlock lang="bash" code={`lunar add search-bar`} />

## Usage

<DynamicCodeBlock lang="tsx" code={`import { SearchBar } from '@/lunar-kit/components/search-bar';`} />

### Basic Usage

<DynamicCodeBlock lang="tsx" code={`const [value, setValue] = React.useState('');

<SearchBar
  placeholder="Search..."
  value={value}
  onValueChange={setValue}
/>`} />

### With Debounced Search

Trigger search after the user stops typing:

<DynamicCodeBlock lang="tsx" code={`<SearchBar
  placeholder="Search products..."
  value={value}
  onValueChange={setValue}
  onDebouncedValueChange={(val) => fetchResults(val)}
  debounceDelay={300}
/>`} />

### Disabled

<DynamicCodeBlock lang="tsx" code={`<SearchBar
  placeholder="Search..."
  disabled
/>`} />

## API Reference

### SearchBar

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | ‚Äî | Controlled value |
| `defaultValue` | `string` | `''` | Default value (uncontrolled) |
| `placeholder` | `string` | `'Search...'` | Placeholder text |
| `onValueChange` | `(value: string) => void` | ‚Äî | Value change callback |
| `onDebouncedValueChange` | `(value: string) => void` | ‚Äî | Debounced value change callback |
| `debounceDelay` | `number` | `500` | Debounce delay in milliseconds |
| `className` | `string` | ‚Äî | Additional classes |
| `disabled` | `boolean` | `false` | Disable the search bar |
